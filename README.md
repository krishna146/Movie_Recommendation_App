# CineMatch: A Data-Driven Movie Recommendation System with Content-Based Filtering and Cross-Platform Android Deployment
This project presents a movie recommendation system built on metadata from approximately 5,000 movies sourced from the publicly available TMDB dataset. The system takes a movie title as input and returns the five most similar movies by analyzing features such as genres, plot keywords, cast, and director. These features are combined into a tag string per movie, converted into numerical vectors using the Bag of Words technique, and compared using cosine similarity to identify the closest matches.
